# BridgeVerse 🌉

Welcome to **BridgeVerse**, an interactive and educational website dedicated to the fascinating world of **Bridge Engineering**. 

## 📖 About the Project
This website was created to share knowledge about bridges and engineering and to help others understand their vital importance in our daily lives. From the history of ancient river crossings to modern high-tech construction, BridgeVerse explores the factors, designs, and science of how bridges connect our world.

## ✨ Features
- **Comprehensive Content:** Navigate through sections covering the History, Types, Real-World Examples, and Notable Failures of bridges.
- **Modern & Interactive UI:** Built with a responsive glassmorphism left-drawer menu, smooth transitions, and a dark/light mode toggle.
- **Anime/Synthwave Aesthetic:** The "About the Author" page features a unique interactive experience with 3D parallax hover effects, falling sakura petals, glowing neon orbs, a moving synthwave grid, and a custom hacker/cyberpunk cursor.

## 🛠️ Technologies Used
- HTML5
- CSS3 (Glassmorphism, Keyframe Animations, Flexbox, Responsive Media Queries)
- Vanilla JavaScript (DOM Manipulation, 3D Parallax Hover Effects)
